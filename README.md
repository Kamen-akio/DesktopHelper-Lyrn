# DesktopHelper Lyrn

Lyrn桌面助手仓库基于Apache-2.0开源。  
运行环境基于 **Windows (x64)** 平台（版本最低支持 **Windows 7**）！  
该以Gdiplus作程序图形显示，以Win32Api作程序内部逻辑依赖。  

开发这个项目目的纯粹是为了方便我调用Cmd来运行软件的习惯，可能对大家来说是有一些些别扭。
